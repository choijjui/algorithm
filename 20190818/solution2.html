2/3 골드바흐의 추측

시간 제한 : 2초 메모리 제한 : 256MB

문제 설명

소수는 1과 자기 자신으로밖에 나누어지지 않는 정수이다.
미해결 문제인 골드바흐의 추측은 2보다 큰 모든 짝수는 두개의 소수의 합으로 표시할 수 있다는 것이다.

우리는 이 추측을 연구해보려고 한다.
2보다 큰 어떤 짝수 int number가 두개의 소수의 합으로 표시가 가능하다면,
표현 할 수 있는 갯수를 리턴하시오. 표현할 수 없다면, -1을 리턴하시오.

이때 하나의 소수를 두번 사용하는 것은 허용된다.

참고 / 제약 사항

  • number는 항상 짝수이다.
  • 4 <= number <= 100,000

테스트 케이스

  1. int number = 4
    리턴(정답): 1

    2 + 2 = 4 이므로 1가지가 있다.

  2. int number = 6
    리턴(정답): 1

    3 + 3 = 6 이므로 1가지가 있다.

  3. int number = 10
    리턴(정답): 2

    10은 3 + 7 과 5 + 5로 표현이 가능하다.

  4. int number = 648
    리턴(정답): 27
  5. int number = 3570
    리턴(정답): 154 0 —>
